# Красивый TUI интерфейс для Dazzer

## ✨ Новые возможности

Фаззер теперь имеет красивый терминальный интерфейс (TUI), созданный с помощью `ratatui`, аналогичный Python версии.

## 🎨 Особенности интерфейса

### Визуальный дизайн
- **Цветная рамка** в розовом цвете (#ff4a96ff) - как в Python версии
- **Центрированное отображение** статистики
- **Обновление в реальном времени** - интерфейс обновляется каждые 100ms
- **Чистый экран** - используется альтернативный экран терминала

### Отображаемая статистика

1. **Основная статистика:**
   - Runtime (время работы) - форматируется как "Xs", "Xm Ys" или "Xh Ym"
   - Total Tests (общее количество тестов)
   - Tests/sec (скорость выполнения тестов)
   - Saved Tests (сохраненные тесты)
   - Max Coverage (максимальное покрытие кода)

2. **Статистика ошибок:**
   - Segmentation Faults (ошибки сегментации)
   - Floating Point Exceptions (исключения с плавающей точкой)
   - Timeouts (таймауты)
   - No Errors (успешные тесты)
   - Unique Return Codes (уникальные коды возврата)

3. **Топ коды возврата:**
   - Показывает 5 наиболее частых кодов возврата
   - С описанием типа ошибки (SIGSEGV, SIGFPE, Timeout, Success, Other)

### Управление

- **'q'** или **ESC** - выход из фаззера
- Интерфейс автоматически обрабатывает ввод в фоновом режиме

## 🚀 Использование

Просто запустите фаззер как обычно:

```bash
cd dazzer-rs
nix-shell -p rustc cargo --run "cargo run"
```

Или используйте скрипт:

```bash
./run.sh
```

## 🔧 Технические детали

- Использует `ratatui` для создания TUI
- Использует `crossterm` для управления терминалом
- Полностью переписан модуль `ui.rs`
- Интегрирован с основным циклом приложения в `app.rs`
- Автоматическая очистка терминала при выходе
